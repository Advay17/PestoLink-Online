What is PestoLink?

Drive your robot wirelessly with PestoLink!

There are two parts of any pestolink connection, a "controller" running in a browser (PestoLink-Mobile or PestoLink-Gamepad), and a robot (PestoLink-Receive or PestoLink-MicroPython)

If you want to use your phone or laptop keyboard to control your robot, use Pestolink-Mobile. If want to use a USB Xbox or PlayStation controller, use PestoLink-Gamepad.

Browser support

Not all browsers on all operating systems support Web Bluetooth.

On Windows, Android, Mac, Linux, and ChromeOS, The recommended browser to use is Google Chrome.

On iOS, use the Bluefy app. It is a minimal web browser that supports Web Bluetooth.

Robot support

There are two embedded platforms that PestoLink currently supports, Arduino and MicroPython. The current code bases are designed for ESP32 and RP2040 on XRP Kits respectively, milage may vary on other MCUs.

Arduino based robots use the library PestoLink-Receive.

MicroPython based robots use the module PestoLink-MicroPython.
